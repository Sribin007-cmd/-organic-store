# Organic Fruits and Vegetables Selling Web Application

## Project Overview

This project is a Django-based web application for selling organic fruits and vegetables online. Users can browse products, view details, add items to the cart, place orders, and manage their accounts.

## Technologies Used

* Python
* Django
* SQLite
* HTML
* CSS
* Bootstrap

## Features

* User Registration
* User Login and Logout
* Product Management
* Product Search
* Shopping Cart
* Order Management
* Product Reviews
* Product Recommendations
* Admin Dashboard
* Responsive User Interface

## User Roles

### Admin

* Manage products
* Manage users
* Manage orders

### Staff

* Access admin panel
* Manage assigned data

### User

* Register and login
* Browse products
* Add products to cart
* Place orders

## Database

* SQLite Database
